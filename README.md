# Everwod IA FAQs

Sistema para analizar conversaciones históricas y generar sugerencias automáticas de preguntas frecuentes por empresa.

## Funcionalidades

- Ingesta de conversaciones desde PostgreSQL.
- Separación de mensajes por empresa.
- Filtrado de mensajes no útiles.
- Agrupación semántica de preguntas similares.
- Deduplicación contra FAQs existentes.
- Generación de respuestas sugeridas.
- Validación humana de sugerencias.
- Ejecución automática con scheduler.
